# Ethos AI Environment

Building AI that doesn’t just decide — but thinks, questions, and justifies like a human.

## Overview
This project simulates human-like ethical decision making under uncertainty.

## Features
- Ask questions to reveal hidden information
- Multi-step reasoning
- Memory-based decisions
- Explanation scoring

## Tasks
- Easy: Exam cheating
- Medium: Medical decision
- Hard: Mission vs civilians

## Run
uvicorn app:app --reload

## Test
python inference.py
